## 快速开始

1. **对话功能**：最基础的功能，每次对话消耗5点能量
2. **图像生成**：根据文本描述生成图像
3. **视频生成**：创建动态视频内容
4. **语音合成**：将文本转换为自然语音

## 重要提示

- 所有模型由管理员统一配置，用户无法修改
- 用户可以调整参数设置（temperature、max_tokens等）来优化输出效果
- 参数设置保存在您的个人偏好中，不会影响其他用户
