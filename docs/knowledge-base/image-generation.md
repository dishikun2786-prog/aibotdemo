# 图像生成使用指南

## 功能概述

图像生成功能允许您根据文本描述生成高质量图像。支持两种模式：

- **文生图**：仅根据文字描述生成图像，使用管理员配置的「文生图模型」。
- **图生图**：根据文字描述 + 1～3 张参考图生成或编辑图像，使用管理员配置的「图生图模型」；后端会根据是否上传参考图自动选择对应模型。

在 AI 工作室点击「图像」后，弹窗内可切换「文生图」/「图生图」；图生图模式下必须上传 1～3 张参考图方可生成。

## 当前使用的模型

模型由管理员统一配置。根据当前接入的 AI 提供方，可能包含以下类型：

**MiniMAX 提供方**：
- **image-01**：高质量图像生成模型，产生精细细节。支持文本生成图像和图像生成图像（支持人物主体参考）
- **image-01-live**：实时图像生成模型，支持快速图像生成

**百炼（阿里云 DashScope）提供方**：
- **wanx-v1**：万相文本生成图像 V1，中英文双语，多种风格
- **wan2.6-image**：万相 2.6 图像生成，支持图像编辑与图文混排，可一次生成 1～4 张；**图生图建议使用此模型**
- **qwen-image-max**：千问图像 Max，真实感与文字渲染更强，同步生成，固定 1 张
- **qwen-image-plus**：千问图像 Plus，多艺术风格与图文混排，固定 1 张

管理员可在游戏配置中分别设置「文生图模型」与「图生图模型」；未单独设置时均使用「图像生成模型」。有参考图时后端自动使用图生图模型，无参考图时使用文生图模型。

## 可调整参数

### aspect_ratio（宽高比）

- **可选值**：`1:1`, `16:9`, `9:16`, `4:3`, `3:4`
- **默认值**：`1:1`
- **说明**：图像的宽高比例

**使用建议**：
- 正方形图像：`1:1`（适合头像、图标）
- 横向图像：`16:9`（适合风景、横幅）
- 纵向图像：`9:16`（适合手机壁纸、海报）

### width / height（自定义宽高）

- **范围**：512 - 2048（必须是8的倍数）
- **说明**：自定义图像宽度和高度

**注意**：如果设置了aspect_ratio，则忽略width和height。

### n（生成数量）

- **范围**：1 - 9（MiniMAX）；百炼 wan2.6-image 为 1～4，千问图像模型固定为 1
- **默认值**：1
- **说明**：一次生成多张图像

**使用建议**：
- 一般使用1张即可
- 需要多个选项时，可以生成2-4张（仅部分模型支持）
- 注意：生成多张图像会消耗更多资源

### negative_prompt（反向提示词，百炼/千问）

- **说明**：描述不希望在画面中出现的内容，如模糊、畸形、过饱和等，对画面进行限制
- **长度**：最长 500 字符
- **适用**：wan2.6-image、qwen-image-max、qwen-image-plus 等百炼图像模型

**示例**：`低分辨率，低画质，肢体畸形，手指畸形，画面过饱和，蜡像感，文字模糊扭曲`

### prompt_extend（智能改写提示词，百炼/千问）

- **可选值**：`true`（默认）/ `false`
- **说明**：开启后模型会对正向提示词进行优化与润色，图像细节更丰富；关闭则更可控，需自行优化提示词

### watermark（水印，百炼/千问）

- **可选值**：`true` / `false`（默认）
- **说明**：是否在图像上添加“AI 生成”/“Qwen-Image”等水印标识

### 参考图（图生图/图像编辑）

- **说明**：上传 1～3 张图片作为参考，与文字提示词一起提交，用于图生图或图像编辑。
- **支持模型**：仅 **wan2.6-image** 支持；qwen-image、wanx-v1 等为纯文生图，会忽略参考图。
- **数量**：1～3 张，顺序会保留并传给模型。
- **格式与大小**：JPEG、PNG、BMP、WEBP；单张不超过 10MB；分辨率建议 384～5000 像素。图生图参考图会在前端自动缩放至 384～5000 像素以满足接口要求。
- **使用场景**：在 AI 工作室点击「🖼️ 图像」后，在弹窗中可添加参考图，与描述一起生成。

## 使用示例

### 基础图像生成

```
提示词：一只可爱的白色小猫坐在窗台上，阳光洒在它身上
宽高比：1:1
```

### 风景图像

```
提示词：夕阳下的海滩，海浪轻抚沙滩，天空呈现橙红色渐变
宽高比：16:9
```

### 人物图像

```
提示词：一位穿着未来科技服装的赛博朋克风格角色
宽高比：9:16
```

## 提示词编写技巧

### 1. 详细描述

好的提示词应该包含：
- 主体：要生成的主要对象
- 风格：艺术风格、色彩风格
- 细节：光线、角度、背景
- 情绪：氛围、情感

**示例**：
```
好的提示词：一只橘色的猫咪坐在复古木桌上，温暖的阳光从左侧窗户照入，背景是模糊的书架，柔和的自然光，温馨的氛围

差的提示词：猫
```

### 2. 使用风格关键词

- **艺术风格**：写实、卡通、水彩、油画、素描、赛博朋克
- **色彩风格**：鲜艳、柔和、黑白、暖色调、冷色调
- **光线**：自然光、柔光、逆光、侧光、黄金时刻

### 3. 避免冲突描述

避免在提示词中包含相互矛盾的元素，例如：
- ❌ "白天和夜晚"
- ✅ "黄昏时分"

## 最佳实践

1. **明确主体**：首先描述要生成的主要对象
2. **添加细节**：描述颜色、形状、位置等细节
3. **指定风格**：明确艺术风格和色彩风格
4. **控制比例**：根据用途选择合适的宽高比
5. **迭代优化**：如果结果不满意，调整提示词重新生成

## 常见问题

### Q: 生成的图像质量不高怎么办？

A: 
- 使用更详细的提示词
- 尝试不同的宽高比
- 检查是否使用了正确的模型

### Q: 如何生成特定风格的图像？

A: 在提示词中明确指定风格，例如："赛博朋克风格"、"水彩画风格"等。

### Q: 可以生成人物图像吗？

A: 可以，但需要注意提示词的描述要清晰，避免模糊的描述。

### Q: 一次可以生成多少张图像？

A: MiniMAX 最多 9 张；百炼 wan2.6-image 支持 1～4 张；千问图像（qwen-image-max / qwen-image-plus）固定 1 张。建议按实际需求选择数量。

### Q: 百炼/千问图像支持哪些参数？

A: 宽高比、反向提示词、智能改写提示词、水印均支持；生成数量仅 wan2.6-image 支持多张（1～4），千问图像固定 1 张。

## wan2.6-image（万相 2.6）说明

- **能力**：文生图、图像编辑、图文混排（当前界面以文生图为主）
- **参数**：aspect_ratio 映射为 size（如 1:1→1280×1280，16:9→1280×720），n 为 1～4，支持 negative_prompt、prompt_extend、watermark
- **参考图**：支持传入 1～3 张参考图，与提示词一起用于图生图/图像编辑；在生成图像弹窗中「添加参考图」即可
- **建议**：需要多张备选时可将 n 设为 2～4；反向提示词可写“低分辨率、肢体畸形、文字模糊”等以提升成图质量

## Qwen-Image（千问图像）说明

- **能力**：文生图，擅长复杂文字渲染与多艺术风格；qwen-image-max 真实感更强，qwen-image-plus 风格更丰富
- **参数**：aspect_ratio 映射为固定分辨率（如 1:1→1328×1328，16:9→1664×928），**n 固定为 1**，支持 negative_prompt、prompt_extend、watermark、seed
- **建议**：正向提示词不超过 800 字；需要可复现时可传 seed；关闭 prompt_extend 时请自行把提示词写详细
