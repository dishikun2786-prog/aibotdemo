# 对话功能使用指南

## 功能概述

对话功能是AI智能体的核心功能，允许您与AI智能体进行自然语言对话。AI智能体会根据您的角色设定和形象设定，以个性化的方式与您交流。

## 当前使用的模型

模型由管理员统一配置，当前使用的模型会在设置界面显示。常见模型包括：

- **MiniMax-M2.5**：峰值性能，极致价值，掌握复杂任务（输出速度约60 tps）
- **MiniMax-M2.5-highspeed**：M2.5高速版，相同性能，更快更敏捷（输出速度约100 tps）
- **MiniMax-M2.1**：强大的多语言编程能力，全面提升编程体验（输出速度约60 tps）
- **MiniMax-M2**：智能体能力，高级推理

## 可调整参数

### temperature（温度）

- **范围**：0.01 - 1.0
- **默认值**：0.7
- **说明**：控制输出的随机性和创造性
  - 较低值（0.1-0.3）：输出更加确定性和保守
  - 中等值（0.5-0.7）：平衡创造性和准确性
  - 较高值（0.8-1.0）：输出更加随机和创造性

**使用建议**：
- 需要准确答案时，使用较低值（0.3-0.5）
- 需要创意内容时，使用较高值（0.7-0.9）
- 一般对话推荐使用 0.7

### max_tokens（最大token数）

- **范围**：1 - 204800
- **默认值**：2000
- **说明**：限制AI生成回复的最大长度

**使用建议**：
- 简短回复：500-1000 tokens
- 一般对话：1500-3000 tokens
- 长文本生成：5000-10000 tokens

**注意**：token数不等于字符数，通常1个token约等于0.75个中文字符。

### top_p（核采样参数）

- **范围**：0.0 - 1.0
- **默认值**：0.95
- **说明**：控制模型考虑的概率质量范围
  - 较低值：只考虑最可能的token
  - 较高值：考虑更多可能的token

**使用建议**：
- 一般使用默认值 0.95
- 需要更确定性输出时，可降低到 0.8-0.9
- 需要更多样性时，可提高到 0.95-1.0

## 使用示例

### 基础对话

```
用户：你好，介绍一下你自己
AI：你好！我是你的专属AI智能体...
```

### 角色扮演

通过设置角色和形象，AI会以特定身份与您对话：

```
角色设定：钢铁侠的贾维斯
形象设定：白色小猫猫
```

### 游戏状态感知

AI会感知您的游戏状态（能量、体力、占据节点等），并在对话中自然地提及：

```
AI：检测到您的能量已达到100，建议进行PK对战释放能量。
```

## 最佳实践

1. **明确角色设定**：清晰的角色设定能让AI更好地理解您的期望
2. **合理设置temperature**：根据对话类型调整温度参数
3. **控制回复长度**：使用max_tokens避免生成过长的回复
4. **利用游戏状态**：AI会结合您的游戏状态给出建议
5. **保存对话历史**：重要对话可以查看历史记录

## 常见问题

### Q: 为什么AI回复很简短？

A: 检查max_tokens设置，如果设置过小会导致回复被截断。

### Q: 如何让AI更有创意？

A: 提高temperature值到0.8-0.9，同时可以适当提高top_p。

### Q: 对话消耗多少能量？

A: 每次对话消耗5点能量（可在管理后台配置）。

### Q: 能量不足怎么办？

A: 通过占据节点挖矿或进行PK对战来获取能量。
