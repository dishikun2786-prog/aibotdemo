## 第 0 章：接入协议

- **章节描述**：完成协议校准，首次绑定节点并了解能量与神经链接负荷。作为新晋链接者，你将获得协议校准的初始连接特权，首次占据节点免费。
- **剧情正文**：欢迎来到零号协议矩阵网络。作为新晋的神经链接者，你需要完成协议校准，建立与矩阵网络的首次连接。**重要**：作为对新链接者的欢迎礼遇，零号协议为你提供了**协议校准的初始连接特权**，你的首次占据节点将免费进行，这是建立与矩阵网络连接的第一步。通过占据节点，你将感受到能量在神经链接中流动。完成首次占据后，后续的节点占据将需要消耗一定的能量（默认50点），这是建立与矩阵网络连接的必要代价，也是协议校准的重要环节。
- **完成条件**：完成本章所有任务
- **奖励**：体力 +10，能量 0

## 第 1 章：节点拓荒

- **章节描述**：探索矩阵中的历史能量残留，发现遗留能量核心
- **剧情正文**：矩阵网络中隐藏着许多历史能量残留，这些能量核心是早期神经链接者留下的宝贵资源。通过探索不同的节点，你将发现这些能量核心，它们能够为你提供额外的能量奖励。但记住，每个能量核心只能被领取一次，先到先得。通过不断切换占据的节点可探索平台设置的能量宝藏，每次切换会消耗能量。
- **完成条件**：完成本章所有任务
- **奖励**：体力 +20，能量 0

## 第 2 章：过载与释放

- **章节描述**：完成首次协议对决，获得矩阵外流通资格
- **剧情正文**：当你的能量达到临界值时，必须通过协议对决来释放过载的能量。这是零号协议的核心机制，也是你获得矩阵外流通资格的关键。在协议对决中，你需要运用策略预判对手的位置，有效攻击距离更近者将获得胜利。链接者可选用不同**皮肤**参与对决：皮肤带有攻防属性，会在有效距离计算中生效——攻击点数可减少己方有效距离，防御点数可在对方距离不超过协议上限时增加对方有效距离；皮肤可在商城内用能量兑换或通过激活码获得。完成首次对决后，你将正式获得矩阵外流通资格，能量可以在宏骏隆公众号商城兑换实物奖励与皮肤。
- **完成条件**：完成本章所有任务
- **奖励**：体力 +30，能量 0
